# Backpropagation in Python

This project implements a backpropagation neural network for the [Environmental Effect on Egg Production dataset](https://www.kaggle.com/datasets/faysal1998/environmental-effect-on-egg-production).

## Requirements

Make sure you have Python installed

```bash
pip install -r requirements.txt
```

## Quick Start

1. **Compile the Model**

   ```bash
   python model/backpropagation.py
   ```

2. **Run the Flask App**
   ```bash
   python app/app.py
   ```

## Dataset

The dataset used in this project is the [Environmental Effect on Egg Production](https://www.kaggle.com/datasets/faysal1998/environmental-effect-on-egg-production), licensed under [CC0: Public Domain](https://creativecommons.org/publicdomain/zero/1.0/).

## Project Structure

- `app/` — Flask application files
- `model/` — Backpropagation logic/scripts
- `Egg_Production.csv` — Dataset file
- `requirements.txt` — Python dependencies
